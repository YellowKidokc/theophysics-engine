# Theophysics Engine

  An advanced axiom modeling and simulation interface for the Theophysics framework.

  ## Features
  - 188 axioms displayed in an interactive chain
  - Deep drill-down into formal statements, common sense explanations
  - Cross-domain mappings (physics, theology, consciousness, quantum mechanics)
  - Judge & Jury courtroom analysis sections
  - Interactive dependency graphs
  - Dark sci-fi aesthetic (deep space blue palette)

  ## Tech Stack
  - **Frontend**: React + Tailwind CSS + shadcn/ui
  - **Backend**: Express.js API with PostgreSQL (Drizzle ORM)
  - **Routing**: wouter

  ## Getting Started
  ```bash
  npm install
  npm run dev
  ```
